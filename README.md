## 📌 Project Overview

This project implements a **Spelling Correction System** using the **Jaccard Similarity** algorithm in Python.

The system compares misspelled words with a dictionary of valid words and suggests the closest matching correction based on character similarity.

It demonstrates one of the fundamental approaches used in **Natural Language Processing (NLP)** for text preprocessing and spell checking.

## 🧠 What is Jaccard Similarity?

Jaccard Similarity measures how similar two sets are.

### Formula

```
J(A,B) = |A ∩ B| / |A ∪ B|
```

Where:

- **A** = Characters of the input word
- **B** = Characters of the dictionary word

A higher similarity score indicates a better spelling match.
